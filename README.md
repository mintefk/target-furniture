# Target Furniture — Luxury E‑commerce Starter

Modern luxury e‑commerce frontend for **Target Furniture**, built with:

- **Next.js 14 (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **shadcn-style UI primitives**
- **Framer Motion** for subtle animation

## Available pages

- **Home** — hero, featured products, categories, story, testimonials, newsletter
- **Shop** — product grid with category filter, search, and sort
- **Product detail** — large gallery, details, add to cart, related products
- **Cart** — cart items, quantity controls, price summary
- **About** — brand story, mission, craftsmanship
- **Contact** — contact form, studio info, social links

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
